# 쉬는시간 게임실

고등학생들이 쉬는시간에 바로 플레이할 수 있는 브라우저 미니게임 사이트입니다.

## 폴더 구조

```text
breaktime-games/
  index.html
  games/
    reaction/
      index.html
      style.css
      script.js
    omok/
      index.html
      style.css
      script.js
    memory/
      index.html
      style.css
      script.js
  assets/
    images/
    sounds/
  README.md
```

## 게임 추가 방식

새 게임은 `games/새게임이름/` 폴더를 만들고 `index.html`, `style.css`, `script.js`를 넣으면 됩니다.
공통 이미지와 사운드는 `assets/images/`, `assets/sounds/`에 보관합니다.
